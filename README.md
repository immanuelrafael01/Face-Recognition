## Project Overview
This project implements a real-time face recognition system using Computer Vision and Deep Learning techniques.

## Technologies
- Python
- OpenCV
- TensorFlow (CNN)
- Streamlit

## Features
- Face detection using Haar Cascade
- Face recognition using CNN
- Real-time webcam input

## Dataset
Custom dataset collected using webcam. Each class contains ~100 images.
